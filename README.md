Abonelik Takip Sistemi (Subscription Tracker)

Bu proje, kullanıcıların aylık aboneliklerini (Netflix, Spotify vb.) tek bir panelden yönetmelerini, harcamalarını grafiklerle analiz etmelerini sağlayan modern bir **SaaS** uygulamasıdır.

Özellikler
**Gerçek Zamanlı Veri:** Supabase ile anlık veri yönetimi.
**Dinamik Grafikler:** Recharts kullanarak aylık harcama ve kategori bazlı analiz.
**Otomatik Logo:** Clearbit API entegrasyonu ile girilen platformun logosunu otomatik çekme.
**Server Actions:** Next.js 14'ün en güncel veri işleme yöntemleri.
**Güvenli Giriş:** Supabase Auth ile e-posta/şifre doğrulaması.

Kullanılan Teknolojiler
**Framework:** Next.js 14 (App Router)
**Veritabanı & Auth:** Supabase
**Stil:** Tailwind CSS
**Grafik Kütüphanesi:** Recharts
**Dil:** TypeScript
**Deployment:** Vercel
