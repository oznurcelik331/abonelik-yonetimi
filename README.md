Abonelik Takip & Finansal Analiz Sistemi

Bu proje, kullanıcıların dijital aboneliklerini (Netflix, Spotify, YouTube vb.) tek bir merkezden yönetmelerini, harcamalarını grafiklerle analiz etmelerini sağlayan modern bir **Full-Stack SaaS** uygulamasıdır.

Öne Çıkan Özellikler
**Dinamik Veri Yönetimi:** Supabase (PostgreSQL) ile güvenli veri depolama ve gerçek zamanlı güncelleme.
**Finansal Görselleştirme:** Recharts kütüphanesi ile harcama trendleri ve kategori bazlı pasta grafik analizleri.
**Akıllı Logo Entegrasyonu:** Clearbit API kullanarak, girilen abonelik ismine göre şirket logosunu otomatik getirme.
**Modern Teknoloji Yığını:** Next.js 14 Server Actions ile API katmanına gerek kalmadan performanslı veri işleme.
**Güvenli Kimlik Doğrulama:** Supabase Auth ile kullanıcı bazlı özel oturum yönetimi.

Teknik Yetkinlikler (Tech Stack)
**Frontend:** Next.js 14 (App Router), TypeScript, Tailwind CSS
**Backend/DB:** Supabase, PostgreSQL
**Grafik/Analiz:** Recharts
**API:** Clearbit Logo API
**Yayın (Deployment):** Vercel
